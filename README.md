# MYSITE

Benvenuto su **MYSITE**, il sito personale di Matteo.  
Un portfolio multi-pagina, minimalista, elegante e “billionaire-style”, costruito per mostrare competenze in musica, programmazione, tecnologia e design.

---

## 🔗 Struttura del progetto

MYSITE/
│
├─ index.html # Homepage con hero e overview
├─ piano.html # Sezione Pianoforte
├─ code.html # Sezione Web & Code
├─ tech.html # Sezione Tecnologia
├─ design.html # Sezione Design & Ideas
├─ style.css # Foglio di stile principale
├─ img/ # Cartella immagini
│ └─ profile.jpg
└─ README.md


---

## 💻 Tecnologie utilizzate

- HTML5  
- CSS3 (Flexbox, Grid, Animazioni base)  
- Formspree per il form contatti  
- Font: Inter (Google Fonts)  
- Gradiente e design minimal dark, stile moderno e professionale  

---

## 🎨 Caratteristiche principali

- **Hero section** con titolo, sottotitolo e foto profilo  
- **Navigazione pulita** e fixed top  
- **Cards cliccabili** che portano a sezioni dedicate o pagine separate  
- **Multi-page layout**: Pianoforte, Code, Technology, Design & Ideas  
- **Form contatti funzionante** tramite Formspree  
- Design **responsive** per mobile e desktop  
- Effetti hover eleganti su link e card
